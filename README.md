# Coin Escape

A simple Pygame-based stealth game where you collect coins while avoiding an enemy that gets faster as your score increases.

## How to Play

- Run the file `main.py`
- Move with arrow keys.
- Press SPACE to enter "dark mode" for 5 seconds: coins become visible, but the enemy sees you and pursues aggressively.
- Collect coins by walking over them in dark mode.
- Reach the door (top-right) to advance to a new level.
- Avoid the enemy; collision resets your score to 0.
- Highscore is tracked and saved when you exit through the door.

## Requirements

- Python 3.x
- Pygame library (`pip install pygame`)


Enjoy the game!
